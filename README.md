# FileFetcher
Fetches js, json, php, txt file from a list of URLS from waybackurls output.

## prerequisite
- [waybackurls](https://github.com/tomnomnom/waybackurls)

## Usage
### To run  
```sh
./ffrun.sh <target>
```

### To export results to files
```sh
./ffrun.sh <target> -o
./ffrun.sh urls.txt -o
```
